南昌 ​

第一天 ​

行程安排

当前位置 → 南昌 → 万寿宫（珠宝街） → 秋水广场 → 八一大桥

路线

居住🏪：新万达广场附近（位置较居中，附近商城较多，出行方便，具体酒店视情况而定）需提前10天或者1周左右预定。

  1. 可先游玩之后再到小吃街吃东西
    1. 南昌水煮
    2. 皮蛋瓦罐汤
    3. 拉面麻油小包子
    4. 老太婆辣藕
    5. 南昌拌粉
    6. 串串
    7. 麻糍
    8. 糯叽叽的热罩子
    9. 腊肠蛋黄糯米饭
    10. 糊羹
    11. 芋泥麻糍
    12. 竹筒奶茶（鲲茶）
    13. 绿豆饼
    14. 卷满肉泥的花卷
  2. 晚上可以去看喷泉 (第一场：19:30 - 19:45、第二场：20:30 - 20:45) 具体时间得再查
    1. 景点位置：江西省南昌市东湖区雄州路与赣江中大道交叉口
  3. 散步有点长，视情况而定

第二天 ​

行程安排

江西省博物馆 → 羊子巷 → 滕王阁游轮 → 新万达

路线

  1. 。
    1. 周一闭馆（法定节假日除外）
    2. 开放时间：9:00 - 17:00
    3. 需要携带
    4. 小程序：进行预约
    5. 景点位置：1号线珠江路口站
    1. 小吴水煮
    2. 芋泥麻糍（有相同）
    3. 麻辣肠子、麻辣莲藕
    4. 鸡蛋汉堡
    5. 汤泡饭
    6. 粉蒸肉
    7. 咸蛋黄糯米饭（有相同）
    8. 咸菜笋馅的包子
    9. 皮薄馅大的烧饼
    10. 甜豆花
    11. 野生小板栗
    12. 韭菜大饼
  3. 夜间游轮。
    1. 景点位置：江西省南昌市东湖区仿古街58号
    2. 取票地点：东湖区榕门路仿古街76号，滕王阁游轮售票处
    3. 时间：晚间第一场：19:00 - 20:45
    4. 购票：南昌本地宝：进行购票 -
  4. 可以购物和看电影。

第三天 ​

行程安排

第一种：八一起义纪念馆 → 绳金塔 → 699文化创意园 → 象湖公园（晚上） → 国贸（晚上）
第二种：八一起义纪念馆 → 动物园
第三种：八一起义纪念馆 → 国贸

路线

  1. 。
    1. 周一闭馆（法定节假日除外）
    2. 开放时间：9:00 - 17:00
    3. 需要携带
    4. 小程序：进行预约
    5. 景点位置：地铁1号线八一馆站
  2. 之后可以在绳金塔美食街吃东西。
  3. 自驾40min左右（开放时间：8:30 - 18:30、需要携带、小程序：进行买票）
  4. 适合拍照。
    1. 景点位置：南昌市青云谱区施尧路
